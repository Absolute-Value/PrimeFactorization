# 素因数分解を行うプログラム

個別塾のバイト中に素因数分解は繰り返しのため、コンピュータが得意そうと感じたため実装しました

## python
PrimeFactorization.py
```console
$ python3 PrimeFactorization.py
素因数分解したい整数を入力してください：1024
[2, 2, 2, 2, 2, 2, 2, 2, 2, 2]
```

## java
PrimeFactorization.java
```console
$ javac PrimeFactorization.java
$ java PrimeFactorization
素因数分解したい整数を入力してください：1024
[2, 2, 2, 2, 2, 2, 2, 2, 2, 2]
```
